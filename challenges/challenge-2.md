# Challenge 2: Create Your Own Blog Site With GitHub Pages

> This challenge is not intended to be completed in the workshop time.

# Instructions

As we covered in the workshop the [blog feed](https://notanother.pizza/blog-workshop/) is generated using a static site generator, and hosted using GitHub Pages. Whilst not all blogs are published in this way this is a relatively easy (and free) way to set up your own.

For this challenge create a repo in your GitHub account, set up a static site generator project (Hugo or Jekyll are both good options), create your first blog post, and host the whole site using GitHub Pages.

A good place to start is to check out the [GitHub Pages documentation](https://docs.github.com/en/pages) and to fork a blog website template:

- *[Jekyll Blog Templates](https://jekyllrb.com/docs/themes/)*
- *[Hugo Blog Templates](https://themes.gohugo.io/tags/blog/)*

For a complete guide on creating your own blog with Jekyll and GitHub Pages see [Setting Up a GitHub Pages Site With Jekyll](https://docs.github.com/en/pages/setting-up-a-github-pages-site-with-jekyll/creating-a-github-pages-site-with-jekyll)
